# Ethereum Devcontainers

Almost everything you could need to develop in solidity.

# Features

Offers a vast array of tools:

- Python
- NodeJS
- vim 
- wget
- solc
- Myril
- solc-select
- Crytic-compile
- Manticore
- Slither

also:
- Colorful `.bashrc` with `vi`
- `.inputrc` for indicating `vi` status
- some lovely scripts under `/scripts`
